# github-readme-optimizer (OpenClaw skill)

Audit and rewrite GitHub README files to maximize discoverability by AI agents/LLMs and human developers.

## What this repo contains

- `SKILL.md` — the OpenClaw skill definition + workflow.

## Install (local)

```bash
mkdir -p ~/.openclaw/skills
cd ~/.openclaw/skills
git clone https://github.com/gaoryan86/openclaw-skill-github-readme-optimizer github-readme-optimizer
```

Restart/reload your OpenClaw agent if needed.

## Usage

In chat, ask things like:

- “optimize README for https://github.com/OWNER/REPO”
- “review readme / README SEO”
- “让 AI 找得到：帮我优化这个项目的 README”
